# 时间戳工具功能文档

## 功能概述
时间戳工具是一个时间转换工具，提供当前时间实时显示、时间戳与日期时间的双向转换、相对时间计算等功能。支持秒级和毫秒级时间戳，自动识别时间戳类型，并提供本地时间和UTC时间的显示。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **JavaScript Date对象**: 时间处理核心API
  - `new Date()`: 创建日期对象
  - `Date.now()`: 获取当前时间戳（毫秒）
  - `getTime()`: 获取时间戳
  - `toLocaleString()`: 本地时间格式化
  - `toUTCString()`: UTC时间格式化
  - `toISOString()`: ISO 8601格式
- **定时器**: setInterval实现实时时钟

### 关键组件
- **Element Plus**: UI组件库
  - `el-input`: 时间戳输入
  - `el-date-picker`: 日期时间选择器
  - `el-button`: 操作按钮

### JavaScript Date API核心方法
```typescript
// 创建日期对象
const date = new Date()              // 当前时间
const date = new Date(timestamp)     // 从时间戳创建
const date = new Date('2024-01-01')  // 从字符串创建

// 获取时间戳
date.getTime()  // 毫秒级时间戳

// 格式化
date.toLocaleString('zh-CN')  // 本地时间: "2024/1/1 00:00:00"
date.toUTCString()            // UTC时间: "Sun, 01 Jan 2024 00:00:00 GMT"
date.toISOString()            // ISO格式: "2024-01-01T00:00:00.000Z"
```

## 实现思路与流程

### 1. 当前时间实时显示

```typescript
const currentTimestamp = ref('')
const currentTimestampMs = ref('')
const currentTime = ref('')
const currentUTCTime = ref('')
let timer: number | null = null

const updateCurrentTime = () => {
  const now = new Date()
  
  // Unix时间戳（秒）
  currentTimestamp.value = Math.floor(now.getTime() / 1000).toString()
  
  // Unix时间戳（毫秒）
  currentTimestampMs.value = now.getTime().toString()
  
  // 本地时间
  currentTime.value = now.toLocaleString('zh-CN')
  
  // UTC时间
  currentUTCTime.value = now.toUTCString()
}

// 组件挂载时启动定时器
onMounted(() => {
  updateCurrentTime()
  timer = window.setInterval(updateCurrentTime, 1000)
})

// 组件卸载时清理定时器
onUnmounted(() => {
  if (timer) {
    clearInterval(timer)
  }
})
```

#### 定时器管理
- **启动**: onMounted中创建，每秒更新一次
- **清理**: onUnmounted中清除，避免内存泄漏
- **类型**: 使用`number`类型（浏览器环境）

### 2. 时间戳转日期

```typescript
const timestampInput = ref('')
const timestampResult = ref<any>(null)

const timestampToDate = () => {
  const ts = parseInt(timestampInput.value)
  
  if (isNaN(ts)) {
    ElMessage.error('请输入有效的时间戳')
    return
  }
  
  // 自动识别秒或毫秒
  const isMilliseconds = ts.toString().length > 12
  const date = new Date(isMilliseconds ? ts : ts * 1000)
  
  timestampResult.value = {
    local: date.toLocaleString('zh-CN'),
    utc: date.toUTCString(),
    iso: date.toISOString(),
    relative: getRelativeTime(date)
  }
}
```

#### 时间戳识别逻辑
```typescript
// 秒级时间戳: 10位数字（到2286年）
// 1704067200 (2024-01-01)

// 毫秒级时间戳: 13位数字
// 1704067200000 (2024-01-01)

// 判断方法: 长度超过12位即为毫秒级
const isMilliseconds = ts.toString().length > 12
```

### 3. 日期转时间戳

```typescript
const dateInput = ref('')
const dateResult = ref<any>(null)

const dateToTimestamp = () => {
  const date = new Date(dateInput.value)
  
  if (isNaN(date.getTime())) {
    ElMessage.error('请选择有效的日期时间')
    return
  }
  
  dateResult.value = {
    seconds: Math.floor(date.getTime() / 1000),
    milliseconds: date.getTime()
  }
}
```

### 4. 填充当前时间

```typescript
// 填充当前时间戳
const fillCurrentTimestamp = () => {
  timestampInput.value = currentTimestamp.value
  timestampToDate()
}

// 填充当前日期时间
const fillCurrentDate = () => {
  const now = new Date()
  // 格式化为 "YYYY-MM-DD HH:mm:ss"
  dateInput.value = now.toISOString()
    .slice(0, 19)
    .replace('T', ' ')
  dateToTimestamp()
}
```

### 5. 相对时间计算

```typescript
const getRelativeTime = (date: Date): string => {
  const now = new Date()
  const diff = now.getTime() - date.getTime()
  const seconds = Math.abs(Math.floor(diff / 1000))
  const minutes = Math.floor(seconds / 60)
  const hours = Math.floor(minutes / 60)
  const days = Math.floor(hours / 24)
  
  if (diff > 0) {
    // 过去时间
    if (days > 365) return `${Math.floor(days / 365)} 年前`
    if (days > 30) return `${Math.floor(days / 30)} 个月前`
    if (days > 0) return `${days} 天前`
    if (hours > 0) return `${hours} 小时前`
    if (minutes > 0) return `${minutes} 分钟前`
    return `${seconds} 秒前`
  } else {
    // 未来时间
    if (days > 365) return `${Math.floor(days / 365)} 年后`
    if (days > 30) return `${Math.floor(days / 30)} 个月后`
    if (days > 0) return `${days} 天后`
    if (hours > 0) return `${hours} 小时后`
    if (minutes > 0) return `${minutes} 分钟后`
    return `${seconds} 秒后`
  }
}
```

#### 相对时间层级
```
秒级: 0-59秒
分钟级: 1-59分钟
小时级: 1-23小时
天级: 1-29天
月级: 1-11个月（按30天计算）
年级: 1+年（按365天计算）
```

### 6. 复制功能

```typescript
const copyToClipboard = (text: string | number) => {
  navigator.clipboard.writeText(text.toString())
  ElMessage.success('已复制')
}
```

## Unix时间戳详解

### 定义
Unix时间戳是从1970年1月1日00:00:00 UTC（Unix纪元）开始经过的秒数。

### 重要时间点
```
0:              1970-01-01 00:00:00 UTC (Unix纪元)
946684800:      2000-01-01 00:00:00 UTC (千禧年)
1000000000:     2001-09-09 01:46:40 UTC (十亿秒纪念日)
1609459200:     2021-01-01 00:00:00 UTC
1704067200:     2024-01-01 00:00:00 UTC
2147483647:     2038-01-19 03:14:07 UTC (32位溢出)
```

### 32位溢出问题
- 32位有符号整数最大值: 2,147,483,647
- 溢出时间: 2038年1月19日03:14:07 UTC
- 解决方案: 使用64位整数或毫秒级时间戳

### 秒vs毫秒
```typescript
// 秒级（10位）
Math.floor(Date.now() / 1000)
// 示例: 1704067200

// 毫秒级（13位）
Date.now()
// 示例: 1704067200000
```
