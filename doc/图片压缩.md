# 图片压缩功能文档

## 功能概述
图片压缩是一个纯前端实现的图片处理工具，支持批量压缩PNG、JPEG、WebP格式的图片，可调节压缩质量、最大宽高尺寸，并实时显示压缩前后的对比数据。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **HTML5 Canvas API**: 图片重绘和压缩的核心技术
- **File API**: 文件读取和处理

### 关键组件
- **Canvas元素**: 用于图片重采样和压缩
  - `drawImage()`: 绘制并调整图片尺寸
  - `toBlob()`: 将Canvas内容转换为Blob对象，支持质量参数
- **Element Plus**: UI组件库
  - `el-slider`: 压缩质量调节
  - `el-input`: 尺寸输入
  - `el-checkbox`: 宽高比选项
- **URL.createObjectURL/revokeObjectURL**: 创建和释放文件预览URL

### Canvas图片压缩原理
```typescript
// 1. 创建Canvas并设置目标尺寸
const canvas = document.createElement('canvas')
canvas.width = newWidth
canvas.height = newHeight

// 2. 绘制图片（自动进行重采样）
const ctx = canvas.getContext('2d')
ctx.drawImage(img, 0, 0, newWidth, newHeight)

// 3. 导出为Blob，指定质量和格式
canvas.toBlob(
  (blob) => { /* 压缩后的图片 */ },
  mimeType, // 'image/jpeg', 'image/png', 'image/webp'
  quality   // 0-1，压缩质量
)
```

## 实现思路与流程

### 1. 文件上传处理

支持拖拽和点击上传两种方式：

```typescript
// 拖拽上传
const handleDrop = (e: DragEvent) => {
  isDragging.value = false

  if (files) {

}

// 点击上传
const handleFileSelect = (e: Event) => {
  const files = (e.target as HTMLInputElement).files
  if (files) {
    addFiles(Array.from(files))
  }
}

// 添加文件并加载图片信息
const addFiles = async (files: File[]) => {
  if (images.value.length + files.length > 10) {
    ElMessage.warning('最多支持 10 张图片')
    return
  }

  for (const file of files) {
    if (!file.type.startsWith('image/')) continue

    try {
      const img = await loadImage(file)
      const id = Math.random().toString(36).substr(2, 9)

      images.value.push({
        id,
        file,
        preview: URL.createObjectURL(file),
        width: img.width,
        height: img.height,
        processing: false,
      })
    } catch (error) {
      ElMessage.error(`加载图片失败: ${file.name}`)
    }
  }
}

// 加载图片获取尺寸
const loadImage = (file: File): Promise<HTMLImageElement> => {
  return new Promise((resolve, reject) => {
    const img = new Image()
    img.onload = () => resolve(img)
    img.onerror = reject
    img.src = URL.createObjectURL(file)
  })
}
```

### 2. 尺寸计算

根据用户设置的最大宽高和宽高比选项计算目标尺寸：

```typescript
const calculateDimensions = (origWidth: number, origHeight: number) => {
  let newWidth = origWidth
  let newHeight = origHeight

  const maxW = parseInt(maxWidth.value) || Infinity
  const maxH = parseInt(maxHeight.value) || Infinity

  if (origWidth > maxW || origHeight > maxH) {
    const ratioW = maxW / origWidth
    const ratioH = maxH / origHeight
    // 保持宽高比时取较小的缩放比例
    const ratio = maintainAspectRatio.value ? Math.min(ratioW, ratioH) : Math.max(ratioW, ratioH)

    newWidth = Math.floor(origWidth * ratio)
    newHeight = Math.floor(origHeight * ratio)
  }

  return { width: newWidth, height: newHeight }
}
```

### 3. 核心压缩逻辑

```typescript
const compressImageFile = async (
  file: File,
  origWidth: number,
  origHeight: number
): Promise<CompressedResult> => {
  return new Promise((resolve, reject) => {
    const img = new Image()
    img.onload = () => {
      // 计算目标尺寸
      const { width, height } = calculateDimensions(origWidth, origHeight)

      // 创建Canvas
      const canvas = document.createElement('canvas')
      canvas.width = width
      canvas.height = height
      const ctx = canvas.getContext('2d')

      if (!ctx) {
        reject(new Error('Canvas context not available'))
        return
      }

      // JPEG需要白色背景（处理透明通道）
      if (file.type === 'image/jpeg') {
        ctx.fillStyle = '#FFFFFF'
        ctx.fillRect(0, 0, width, height)
      }

      // 绘制图片
      ctx.drawImage(img, 0, 0, width, height)

      // 智能格式转换：PNG低质量时转为JPEG
      const mimeType = file.type === 'image/png' && quality.value < 100
        ? 'image/jpeg'
        : file.type

      // 导出压缩后的Blob
      canvas.toBlob(
        (blob) => {
          if (blob) {
            resolve({
              blob,
              url: URL.createObjectURL(blob),
              size: blob.size,
              width,
              height,
            })
          } else {
            reject(new Error('Compression failed'))
          }
        },
        mimeType,
        quality.value / 100 // 质量参数 0-1
      )
    }
    img.onerror = reject
    img.src = URL.createObjectURL(file)
  })
}
```

### 4. 压缩结果展示

计算压缩比例并显示统计信息：

```typescript
const compressImage = async (index: number) => {
  const image = images.value[index]
  if (!image || image.processing) return

  image.processing = true

  try {
    const result = await compressImageFile(image.file, image.width, image.height)
    const originalSize = image.file.size
    const newSize = result.size

    image.result = result
    // 计算压缩比例（正数表示缩小，负数表示增大）
    image.compressionRatio = ((originalSize - newSize) / originalSize) * 100

    ElMessage.success('压缩完成')
  } catch (error) {
    ElMessage.error('压缩失败')
  } finally {
    image.processing = false
  }
}

// 批量压缩
const compressAll = async () => {
  isProcessingAll.value = true
  for (let i = 0; i < images.value.length; i++) {
    if (!images.value[i].result) {
      await compressImage(i)
    }
  }
  isProcessingAll.value = false
  ElMessage.success('批量压缩完成')
}
```

### 5. 统计计算

使用computed实时计算统计数据：

```typescript
// 原始总大小
const totalOriginalSize = computed(() =>
  images.value.reduce((sum, img) => sum + img.file.size, 0)
)

// 压缩后总大小
const totalCompressedSize = computed(() =>
  images.value.filter(img => img.result).reduce((sum, img) => sum + (img.result?.size || 0), 0)
)

// 节省空间比例
const totalSavedRatio = computed(() => {
  const compressed = images.value.filter(img => img.result)
  if (compressed.length === 0) return 0
  const originalSize = compressed.reduce((sum, img) => sum + img.file.size, 0)
  const newSize = compressed.reduce((sum, img) => sum + (img.result?.size || 0), 0)
  return ((originalSize - newSize) / originalSize) * 100
})
```

### 6. 下载功能

```typescript
const downloadImage = (image: ImageItem) => {
  if (!image.result) return

  const link = document.createElement('a')
  link.href = image.result.url
  const ext = image.file.name.split('.').pop() || 'jpg'
  const baseName = image.file.name.replace(/\.[^/.]+$/, '')
  link.download = `${baseName}_compressed.${ext}`
  link.click()
}

// 批量下载（间隔500ms避免浏览器拦截）
const downloadAll = () => {
  const processed = images.value.filter(img => img.result)
  if (processed.length === 0) return

  processed.forEach((image, index) => {
    setTimeout(() => {
      downloadImage(image)
    }, index * 500)
  })

  ElMessage.success('开始批量下载')
}
```

### 7. 内存管理

及时释放ObjectURL避免内存泄漏：

```typescript
const removeImage = (index: number) => {
  const image = images.value[index]
  if (image.result?.url) {
    URL.revokeObjectURL(image.result.url)
  }
  URL.revokeObjectURL(image.preview)
  images.value.splice(index, 1)
}

const clearAll = () => {
  images.value.forEach(img => {
    if (img.result?.url) {
      URL.revokeObjectURL(img.result.url)
    }
    URL.revokeObjectURL(img.preview)
  })
  images.value = []
}
```

## 功能总结

### 核心特性
1. **批量处理**: 最多支持10张图片同时压缩
2. **质量调节**: 1-100%压缩质量可调
3. **尺寸调整**: 支持设置最大宽高，可选择保持宽高比
4. **实时预览**: 显示压缩前后的对比数据
5. **智能格式转换**: PNG低质量时自动转JPEG
6. **统计面板**: 实时显示总体压缩效果
7. **拖拽上传**: 便捷的拖拽上传体验

### 技术亮点
- **纯前端实现**: 无需后端，所有处理在浏览器完成
- **Canvas重采样**: 利用Canvas的drawImage实现高质量图片缩放
- **Blob API**: 使用toBlob导出压缩后的图片
- **内存管理**: 正确使用revokeObjectURL释放资源
- **Promise封装**: 异步图片加载和压缩处理
- **Computed统计**: 响应式数据自动计算统计信息

### 用户体验优化
1. 拖拽区域高亮反馈
2. 处理中显示loading状态
3. 压缩比例用颜色区分（绿色表示缩小，红色表示增大）
4. 批量下载间隔避免浏览器拦截
5. 文件大小格式化显示（B/KB/MB/GB）
