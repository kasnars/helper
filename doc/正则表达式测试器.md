# 正则表达式测试器功能文档

## 功能概述
正则表达式测试器是一个在线正则测试工具，支持实时匹配、高亮显示、捕获组展示，并提供常用正则模板，帮助开发者快速测试和验证正则表达式。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **JavaScript RegExp对象**: 正则表达式引擎
  - `new RegExp(pattern, flags)`: 创建正则对象
  - `matchAll()`: 获取所有匹配结果
  - `exec()`: 执行匹配
- **文本处理**: 字符串分割和高亮渲染

### 关键组件
- **Element Plus**: UI组件库
  - `el-input`: 正则表达式和测试文本输入
  - `el-checkbox`: 正则标志选择
  - `el-tag`: 匹配结果标签
- **v-for + template**: 高亮文本渲染

### 正则标志说明
| 标志 | 名称 | 说明 |
|------|------|------|
| g | global | 全局匹配，查找所有匹配项 |
| i | ignoreCase | 忽略大小写 |
| m | multiline | 多行模式，^和$匹配每行 |
| s | dotAll | 点号匹配包括换行符的所有字符 |
| u | unicode | Unicode模式 |

## 实现思路与流程

### 1. 正则表达式构建

```typescript
const pattern = ref('')
const testString = ref('')
const error = ref('')
const matches = ref<MatchResult[]>([])

const flags = ref({
  global: true,
  ignoreCase: false,
  multiline: false,
  dotAll: false,
  unicode: false,
})

// 生成标志字符串
const flagString = computed(() => {
  let str = ''
  if (flags.value.global) str += 'g'
  if (flags.value.ignoreCase) str += 'i'
  if (flags.value.multiline) str += 'm'
  if (flags.value.dotAll) str += 's'
  if (flags.value.unicode) str += 'u'
  return str
})
```

### 2. 实时匹配逻辑

```typescript
interface MatchResult {
  text: string
  index: number
  groups: Record<string, string>
}

// 监听输入变化，自动执行匹配
watch([pattern, testString, flagString], () => {
  if (!pattern.value || !testString.value) {
    matches.value = []
    error.value = ''
    return
  }
  
  try {
    // 创建正则对象
    const regex = new RegExp(pattern.value, flagString.value)
    
    // 执行匹配
    const results: MatchResult[] = []
    
    if (flags.value.global) {
      // 全局匹配：使用matchAll
      for (const match of testString.value.matchAll(regex)) {
        results.push({
          text: match[0],
          index: match.index,
          groups: match.groups || {}
        })
      }
    } else {
      // 单次匹配：使用exec
      const match = regex.exec(testString.value)
      if (match) {
        results.push({
          text: match[0],
          index: match.index,
          groups: {}
        })
      }
    }
    
    matches.value = results
    error.value = ''
  } catch (e: any) {
    error.value = e.message
    matches.value = []
  }
}, { immediate: true })
```

#### matchAll vs exec
```typescript
// matchAll: 返回Iterator，包含所有匹配
const matches = text.matchAll(/pattern/g)
for (const match of matches) {
  console.log(match[0])  // 完整匹配
  console.log(match[1])  // 第一个捕获组
  console.log(match.index)  // 匹配位置
}

// exec: 每次返回一个匹配，需要循环
const regex = /pattern/g
let match
while ((match = regex.exec(text)) !== null) {
  console.log(match[0])
}
```

### 3. 高亮预览实现

```typescript
const highlightedParts = computed(() => {
  if (!pattern.value || !testString.value) {
    return [{ text: testString.value, matched: false }]
  }
  
  try {
    const regex = new RegExp(pattern.value, flagString.value)
    const parts: Array<{ text: string; matched: boolean }> = []
    
    let lastIndex = 0
    for (const match of testString.value.matchAll(regex)) {
      // 添加匹配前的文本
      if (match.index > lastIndex) {
        parts.push({
          text: testString.value.slice(lastIndex, match.index),
          matched: false
        })
      }
      
      // 添加匹配的文本
      parts.push({
        text: match[0],
        matched: true
      })
      
      lastIndex = match.index + match[0].length
    }
    
    // 添加剩余文本
    if (lastIndex < testString.value.length) {
      parts.push({
        text: testString.value.slice(lastIndex),
        matched: false
      })
    }
    
    return parts
  } catch {
    return [{ text: testString.value, matched: false }]
  }
})
```

#### 模板渲染
```vue
<div class="font-mono whitespace-pre-wrap break-all">

      {{ part.text }}
    </mark>
    <span v-else>{{ part.text }}</span>
  </template>
</div>
```

## 常用正则示例

### 邮箱验证
```regex
^[\w.-]+@[\w.-]+\.\w+$
```
- `^`: 字符串开始
- `[\w.-]+`: 用户名（字母、数字、点、下划线、横线）
- `@`: @符号
- `[\w.-]+`: 域名
- `\.`: 点
- `\w+`: 顶级域名
- `$`: 字符串结束

### 手机号码
```regex
^1[3-9]\d{9}$
```
- `^1`: 以1开头
- `[3-9]`: 第二位3-9
- `\d{9}`: 后面9位数字
- `$`: 字符串结束

### URL匹配
```regex
^https?://[\w.-]+\.[a-z]{2,}(/[^\s]*)?$
```
- `^https?://`: http或https
- `[\w.-]+\.`: 域名
- `[a-z]{2,}`: 顶级域名（至少2个字符）
- `(/[^\s]*)?`: 可选路径
- `$`: 字符串结束
