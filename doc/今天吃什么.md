# 今天吃什么功能文档

## 功能概述
"今天吃什么"是一个帮助用户随机选择菜品的趣味工具，通过转盘动画的方式随机选择午餐或晚餐选项，支持自定义添加菜品、分类管理、权重设置等功能。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **HTML5 Canvas API**: 绘制转盘
- **Pinia**: 状态管理，存储用户自定义的菜品选项

### 关键组件

#### 1. FoodWheel.vue - 转盘组件
- **Canvas元素**: 用于绘制彩色转盘
- **requestAnimationFrame**: 实现流畅的旋转动画
- **缓动函数**: easeOutCubic和easeOutBack实现自然的减速和微弹效果

#### 2. FoodManager.vue - 菜品管理组件
- **Element Plus**: UI组件库
  - `el-input`: 输入框
  - `el-select`: 下拉选择器
  - `el-button`: 按钮
  - `el-tag`: 标签
- **Pinia Store**: 使用`useFoodStore`管理菜品数据

#### 3. FoodResult.vue - 结果展示组件
- **Vue Transition**: 实现模态框的进入/离开动画
- **CSS动画**: 使用animate-bounce实现庆祝动画

## 实现思路与流程

### 1. 转盘绘制

```typescript
const drawWheel = () => {
  if (!canvasRef.value || props.options.length === 0) return
  
  const ctx = canvasRef.value.getContext('2d')
  if (!ctx) return

  const centerX = props.size / 2
  const centerY = props.size / 2
  const radius = props.size / 2 - 10

  ctx.clearRect(0, 0, props.size, props.size)

  const anglePerSegment = (2 * Math.PI) / props.options.length

  props.options.forEach((option, index) => {
    const startAngle = index * anglePerSegment - Math.PI / 2
    const endAngle = (index + 1) * anglePerSegment - Math.PI / 2

    // 绘制扇形区域
    ctx.beginPath()
    ctx.moveTo(centerX, centerY)
    ctx.arc(centerX, centerY, radius, startAngle, endAngle)
    ctx.closePath()
    ctx.fillStyle = option.color || defaultColors[index % defaultColors.length]
    ctx.fill()
    ctx.stroke()

    // 绘制文字（旋转坐标系）
    ctx.save()
    ctx.translate(centerX, centerY)
    ctx.rotate(startAngle + anglePerSegment / 2)
    ctx.textAlign = 'right'
    ctx.fillStyle = '#fff'
    ctx.font = 'bold 14px sans-serif'
    
    const text = `${option.emoji} ${option.name}`
    ctx.fillText(text, radius - 20, 5)
    ctx.restore()
  })

  // 绘制中心圆
  ctx.beginPath()
  ctx.arc(centerX, centerY, 35, 0, 2 * Math.PI)
  ctx.fillStyle = '#fff'
  ctx.fill()
}
```

### 2. 旋转动画实现

使用缓动函数实现自然的减速效果：

```typescript
// 三次方缓出函数
const easeOutCubic = (t: number): number => {
  return 1 - Math.pow(1 - t, 3)
}

// 回弹缓动函数（用于最后的微弹效果）
const easeOutBack = (t: number): number => {
  const c1 = 1.70158
  const c3 = c1 + 1
  return 1 + c3 * Math.pow(t - 1, 3) + c1 * Math.pow(t - 1, 2)
}

const spin = () => {
  if (isSpinning.value || props.options.length === 0) return

  isSpinning.value = true
  
  const duration = 4000 // 4秒
  const startTime = Date.now()
  const startRotation = rotation.value
  const minSpins = 5 // 至少转5圈
  const extraSpins = Math.random() * 3 // 随机额外圈数
  const totalRotation = (minSpins + extraSpins) * 360

  const animate = () => {
    const elapsed = Date.now() - startTime
    const progress = Math.min(elapsed / duration, 1)
    
    // 前90%使用easeOutCubic，后10%使用easeOutBack
    let eased: number
    if (progress < 0.9) {
      eased = easeOutCubic(progress / 0.9) * 0.9
    } else {
      const bounceProgress = (progress - 0.9) / 0.1
      eased = 0.9 + easeOutBack(bounceProgress) * 0.1
    }
    
    rotation.value = startRotation + totalRotation * eased

    if (progress < 1) {
      requestAnimationFrame(animate)
    } else {
      // 动画完成
      isSpinning.value = false
      
      // 计算选中的选项
      const finalRotation = rotation.value % 360
      const anglePerSegment = 360 / props.options.length
      const selectedIndex = Math.floor((360 - finalRotation + anglePerSegment / 2) / anglePerSegment) % props.options.length
      
      emit('select', {
        ...props.options[selectedIndex],
        index: selectedIndex,
      })
    }
  }

  requestAnimationFrame(animate)
}
```

### 3. 选中计算逻辑

转盘停止后，根据最终旋转角度计算指针指向的选项：

```typescript
const finalRotation = rotation.value % 360 // 获取0-360的角度
const anglePerSegment = 360 / props.options.length // 每个选项的角度
// 指针在顶部（270度位置），需要反向计算
const selectedIndex = Math.floor((360 - finalRotation + anglePerSegment / 2) / anglePerSegment) % props.options.length
```

## 整体实现细节

### 组件结构

```
FoodPicker.vue (父组件)
├── FoodManager.vue (菜品管理)
│   ├── 添加新选项表单
│   ├── 分类过滤器（全部/午餐/晚餐）
│   └── 选项列表网格
├── FoodWheel.vue (转盘)
│   ├── Canvas画布
│   ├── 旋转动画
│   └── 开始按钮
└── FoodResult.vue (结果弹窗)
    ├── 庆祝动画
    ├── 结果显示
    └── 操作按钮（确定/再选一次）
```

### 数据流

1. **初始化**: FoodStore从IndexedDB加载内置和用户自定义的菜品
2. **管理**: 用户通过FoodManager添加/删除菜品，数据同步到FoodStore
3. **选择**: 用户在FoodWheel点击开始，转盘旋转
4. **结果**: 动画结束后emit选中事件，显示FoodResult弹窗

### 状态管理（Pinia Store）

```typescript
// FoodStore核心功能
interface FoodOption {
  id?: number
  name: string
  emoji: string
  category: 'lunch' | 'dinner'
  weight: number
  isBuiltin: boolean
  color?: string
}

// 主要方法
- loadOptions(): 从数据库加载选项
- addOption(option): 添加新选项
- deleteOption(id): 删除选项
- updateOption(id, updates): 更新选项
```

### 样式实现
- **Tailwind CSS**: 响应式布局和工具类
- **CSS动画**: animate-bounce、backdrop-blur等
- **渐变**: bg-gradient-to-br实现渐变色
- **暗色模式**: dark:前缀支持主题切换

## 功能总结

### 核心特性
1. **趣味转盘**: Canvas绘制的彩色转盘，流畅的旋转动画
2. **缓动效果**: easeOutCubic + easeOutBack组合，实现自然减速和微弹
3. **分类管理**: 支持午餐/晚餐分类，可自定义添加删除
4. **结果展示**: 精美的模态框动画，庆祝效果
5. **数据持久化**: 使用IndexedDB存储用户自定义选项
6. **响应式设计**: 适配不同屏幕尺寸

### 技术亮点
- **纯Canvas绘制**: 不依赖第三方图表库，自定义程度高
- **requestAnimationFrame**: 流畅的60fps动画
- **数学计算**: 精确的角度计算和选中判断
- **缓动函数**: 专业动画效果实现
- **组件化设计**: 三个子组件各司其职，通过props和emits通信
- **状态管理**: Pinia实现全局状态管理

### 用户体验优化
1. 转盘旋转时禁用按钮防止重复点击
2. 动画完成后自动计算并展示结果
3. 支持"再选一次"快速重选
4. 分类过滤快速定位
5. 自定义Emoji增加趣味性
