# 密码管理功能文档

## 功能概述
密码管理是一个本地化的密码存储工具，支持密码的增删改查、搜索过滤、导入导出等功能。所有数据存储在浏览器IndexedDB中，不上传到服务器，保证密码安全。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **IndexedDB**: 浏览器本地数据库
  - 通过Dexie.js封装，简化操作
  - 支持CRUD操作
  - 数据持久化存储
- **Pinia**: 状态管理，管理密码列表

### 关键组件
- **Element Plus**: UI组件库
  - `el-input`: 搜索和密码输入
  - `el-dialog`: 添加/编辑对话框
  - `el-form`: 表单验证
  - `el-button`: 操作按钮
- **Clipboard API**: 复制密码到剪贴板
- **File API**: 导入导出JSON文件

### IndexedDB vs LocalStorage
| 特性 | IndexedDB | LocalStorage |
|------|-----------|--------------|
| 容量 | 大容量（通常50MB+） | 小容量（5-10MB） |
| 查询 | 支持索引和查询 | 仅键值对 |
| 异步 | 异步操作 | 同步操作 |
| 数据类型 | 支持复杂对象 | 仅字符串 |
| 性能 | 高性能 | 大数据时性能差 |

## 实现思路与流程

### 1. 数据库设计

```typescript
// db/index.ts
import Dexie, { Table } from 'dexie'

export interface Password {
  id?: number
  platform: string      // 平台名称
  username: string      // 账号
  password: string      // 密码
  notes?: string        // 备注
  createdAt: Date       // 创建时间
  updatedAt: Date       // 更新时间
}

export class PasswordDB extends Dexie {
  passwords!: Table<Password>

  constructor() {
    super('passwordDB')
    this.version(1).stores({
      passwords: '++id, platform, createdAt'  // 索引字段
    })
  }
}

export const db = new PasswordDB()
```

### 2. Pinia状态管理

```typescript
// stores/index.ts
import { defineStore } from 'pinia'
import { db, Password } from '../db'

export const usePasswordStore = defineStore('password', {
  state: () => ({
    passwords: [] as Password[],
  }),
  
  actions: {
    // 加载所有密码
    async loadPasswords() {
      this.passwords = await db.passwords.toArray()
    },
    
    // 添加密码
    async addPassword(password: Omit<Password, 'id' | 'createdAt' | 'updatedAt'>) {
      const now = new Date()
      const id = await db.passwords.add({
        ...password,
        createdAt: now,
        updatedAt: now,
      })
      await this.loadPasswords()
      return id
    },
    
    // 更新密码
    async updatePassword(id: number, updates: Partial<Password>) {
      await db.passwords.update(id, {
        ...updates,
        updatedAt: new Date(),
      })
      await this.loadPasswords()
    },
    
    // 删除密码
    async deletePassword(id: number) {
      await db.passwords.delete(id)
      await this.loadPasswords()
    },
    
    // 批量导入
    async importPasswords(passwords: Password[]) {
      await db.passwords.bulkAdd(passwords)
      await this.loadPasswords()
    },
    
    // 导出所有密码
    async exportPasswords(): Promise<Password[]> {
      return await db.passwords.toArray()
    },
  },
})
```

### 4. 搜索过滤

```typescript
const searchQuery = ref('')

const filteredPasswords = computed(() => {
  if (!searchQuery.value) {
    return passwordStore.passwords
  }
  
  const query = searchQuery.value.toLowerCase()
  return passwordStore.passwords.filter(item => 
    item.platform.toLowerCase().includes(query) ||
    item.username.toLowerCase().includes(query)
  )
})
```

### 5. 添加/编辑密码

```typescript
const showAddDialog = ref(false)
const editingId = ref<number | null>(null)
const form = ref({
  platform: '',
  username: '',
  password: '',
  notes: '',
})

const editPassword = (item: Password) => {
  editingId.value = item.id || null
  form.value = {
    platform: item.platform,
    username: item.username,
    password: item.password,
    notes: item.notes || '',
  }
  showAddDialog.value = true
}

const savePassword = async () => {
  if (editingId.value) {
    await passwordStore.updatePassword(editingId.value, form.value)
    ElMessage.success('更新成功')
  } else {
    await passwordStore.addPassword(form.value)
    ElMessage.success('添加成功')
  }
  showAddDialog.value = false
  resetForm()
}

const resetForm = () => {
  editingId.value = null
  form.value = {
    platform: '',
    username: '',
    password: '',
    notes: '',
  }
}
```

### 6. 导入导出功能

```typescript
// 导出
const handleExport = async () => {
  const passwords = await passwordStore.exportPasswords()
  const json = JSON.stringify(passwords, null, 2)
  const blob = new Blob([json], { type: 'application/json' })
  const url = URL.createObjectURL(blob)
  
  const link = document.createElement('a')
  link.href = url
  link.download = `passwords_${new Date().toISOString().split('T')[0]}.json`
  link.click()
  
  URL.revokeObjectURL(url)
  ElMessage.success('导出成功')
}

// 导入
const handleImport = () => {
  const input = document.createElement('input')
  input.type = 'file'
  input.accept = '.json'
  
  input.onchange = async (e) => {
    const file = (e.target as HTMLInputElement).files?.[0]
    if (!file) return
    
    const reader = new FileReader()
    reader.onload = async (e) => {
      try {
        const passwords = JSON.parse(e.target?.result as string)
        await passwordStore.importPasswords(passwords)
        ElMessage.success('导入成功')
      } catch {
        ElMessage.error('导入失败，文件格式错误')
      }
    }
    reader.readAsText(file)
  }
  
  input.click()
}
```

## 安全考虑

### 本地存储安全
1. **数据隔离**: IndexedDB同源策略，其他网站无法访问
2. **不上传服务器**: 所有数据存储在本地
3. **HTTPS要求**: 生产环境使用HTTPS

### 潜在风险
1. **XSS攻击**: 如果网站存在XSS漏洞，可能读取IndexedDB
2. **物理访问**: 他人使用你的电脑可能访问数据
3. **浏览器扩展**: 恶意扩展可能读取数据

### 改进建议
1. 添加主密码加密
2. 支持自动锁定（超时后清除内存）
3. 添加密码强度检测
4. 支持生成随机密码
5. 添加两步验证
