# 二维码工具功能文档

## 功能概述
二维码工具是一个双模式工具，支持生成和识别二维码。生成模式支持自定义尺寸、颜色、纠错级别，并提供快速模板（WiFi、电话、邮件、网址）。识别模式支持从图片文件识别和摄像头实时扫描。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **qrcode库**: 用于生成二维码
- **jsQR库**: 用于识别二维码
- **HTML5 Canvas API**: 二维码渲染和图像数据处理
- **MediaDevices API**: 摄像头访问和控制

### 关键组件

#### 1. qrcode库
- **功能**: 将任意文本转换为二维码图像
- **实现原理**: 基于QR Code标准(ISO/IEC 18004)实现编码算法
- **核心API**: 
  ```typescript
  QRCode.toCanvas(canvas, text, options)
  ```

#### 2. jsQR库
- **功能**: 从图像数据中检测和识别二维码
- **实现原理**: 使用图像处理算法定位二维码的三个定位点，然后解码
- **核心API**:
  ```typescript
  jsQR(data, width, height) // 返回二维码数据
  ```

#### 3. Element Plus UI组件
- `el-radio-group`: 模式切换
- `el-slider`: 尺寸调节
- `el-color-picker`: 颜色选择
- `el-form`: 表单布局

### Canvas 图像处理原理

## 实现思路与流程

### 1. 二维码生成

#### 基础生成逻辑
```typescript
const generateQR = async () => {
  if (!qrContent.value) return
  
  await nextTick()
  const canvas = document.createElement('canvas')
  
  try {
    await QRCode.toCanvas(canvas, qrContent.value, {
      width: qrSize.value,
      margin: 2,
      color: {
        dark: qrColor.value,      // 前景色
        light: qrBgColor.value,   // 背景色
      },
      errorCorrectionLevel: qrErrorLevel.value,
    })
    
    qrCanvas.value = canvas
    qrDataUrl.value = canvas.toDataURL('image/png')
  } catch (error) {
    ElMessage.error('生成失败')
  }
}
```

#### 参数配置
- **尺寸**: 100-500px，步长50px
- **颜色**: 前景色和背景色可自定义
- **纠错级别**: 
  - L (Low): 7%容错
  - M (Medium): 15%容错
  - Q (Quartile): 25%容错
  - H (High): 30%容错

#### 快速模板
```typescript
const applyTemplate = (type: string) => {

    url: 'https://www.example.com',

```

```typescript
// 复制到剪贴板
const copyQR = async () => {
  if (!qrCanvas.value) return
  try {
    const blob = await new Promise<Blob>((resolve) => {
      qrCanvas.value!.toBlob((b) => resolve(b!), 'image/png')
    })
    await navigator.clipboard.write([
      new ClipboardItem({ 'image/png': blob })
    ])
    ElMessage.success('已复制到剪贴板')
  } catch {
    ElMessage.error('复制失败')
  }
}
```

### 3. 图片文件识别

```typescript
const processImage = (file: File) => {
  scanResult.value = ''
  scanError.value = ''
  
  const reader = new FileReader()
  reader.onload = (e) => {
    const img = new Image()
    img.onload = () => {
      previewImage.value = e.target?.result as string
      scanImage(img)
    }
    img.src = e.target?.result as string
  }
  reader.readAsDataURL(file)
}

const scanImage = (img: HTMLImageElement) => {
  const canvas = document.createElement('canvas')
  canvas.width = img.width
  canvas.height = img.height
  const ctx = canvas.getContext('2d')
  if (!ctx) return
  
  // 绘制图像到Canvas
  ctx.drawImage(img, 0, 0)
  
  // 获取像素数据
  const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height)
  
  // 识别二维码
  const code = jsQR(imageData.data, imageData.width, imageData.height)
  
  if (code) {
    scanResult.value = code.data
  } else {
    scanError.value = '未能识别二维码，请尝试其他图片'
  }
}
```

### 4. 摄像头实时扫描

#### 启动摄像头
```typescript
const startCamera = async () => {
  try {
    // 请求后置摄像头
    const stream = await navigator.mediaDevices.getUserMedia({ 
      video: { facingMode: 'environment' } 
    })
    
    if (videoRef.value) {
      videoRef.value.srcObject = stream
      isCameraActive.value = true
      scanResult.value = ''
      scanError.value = ''
      
      // 开始定时扫描（每500ms一次）
      scanInterval = window.setInterval(() => {
        scanFromVideo()
      }, 500)
    }
  } catch {
    ElMessage.error('无法访问相机')
  }
}
```

#### 停止摄像头
```typescript
const stopCamera = () => {
  // 停止所有轨道
  if (videoRef.value?.srcObject) {
    const stream = videoRef.value.srcObject as MediaStream
    stream.getTracks().forEach(track => track.stop())
    videoRef.value.srcObject = null
  }
  
  // 清除扫描定时器
  if (scanInterval) {
    clearInterval(scanInterval)
    scanInterval = null
  }
  
  isCameraActive.value = false
}
```

#### 从视频帧扫描
```typescript
const scanFromVideo = () => {
  if (!videoRef.value || !scanCanvas.value) return
  
  const video = videoRef.value
  const canvas = scanCanvas.value
  const ctx = canvas.getContext('2d')
  
  if (!ctx || video.readyState !== video.HAVE_ENOUGH_DATA) return
  
  // 将视频帧绘制到Canvas
  canvas.width = video.videoWidth
  canvas.height = video.videoHeight
  ctx.drawImage(video, 0, 0)
  
  // 获取图像数据并识别
  const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height)
  const code = jsQR(imageData.data, imageData.width, imageData.height)
  
  if (code) {
    scanResult.value = code.data
    stopCamera() // 识别成功后停止扫描
  }
}
```

### 5. 结果处理

```typescript
// 复制识别结果

// 判断是否为URL
const isUrl = (text: string) => {
  return /^https?:\/\//.test(text)
}

// 打开链接
const openScanResult = () => {
  if (isUrl(scanResult.value)) {
    window.open(scanResult.value, '_blank')
  }
}
```

```typescript
const handleDrop = (e: DragEvent) => {
  const file = e.dataTransfer?.files[0]
  if (file) processImage(file)
}

const handleFileSelect = (e: Event) => {
  const file = (e.target as HTMLInputElement).files?.[0]
```

## 功能总结

### 核心特性
1. **双向功能**: 生成和识别二维码
2. **高度定制**: 尺寸、颜色、纠错级别可调
3. **快速模板**: WiFi、电话、邮件、网址一键生成
4. **多源识别**: 支持图片文件和摄像头实时扫描
5. **智能识别**: 自动检测URL并提供打开链接功能
6. **便捷操作**: 一键下载、复制二维码

### 技术亮点
- **第三方库集成**: qrcode生成 + jsQR识别
- **Canvas图像处理**: 像素级数据操作
- **MediaDevices API**: 摄像头访问和视频流处理
- **定时器扫描**: 500ms间隔实现准实时识别
- **Clipboard API**: 现代浏览器剪贴板操作
- **资源管理**: 正确释放摄像头和定时器

### 用户体验优化
