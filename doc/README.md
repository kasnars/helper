## 🏗️ 技术架构

### 前端技术栈
- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **UI组件库**: Element Plus
- **样式**: Tailwind CSS
- **状态管理**: Pinia
- **路由**: Vue Router
- **构建工具**: Vite
- **数据库**: IndexedDB (通过 Dexie.js)

### 核心依赖库
- `qrcode`: 二维码生成
- `jsQR`: 二维码识别
- `crypto.subtle`: Web加密API（哈希计算）
