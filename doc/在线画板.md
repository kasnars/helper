# 在线画板功能文档

## 功能概述
在线画板是一个基于HTML5 Canvas的绘图工具，支持多种绘图工具、颜色选择、画笔粗细调节、撤销/重做、清空画布以及图片导出等功能。

## 关键技术栈

### 核心技术
- **Vue 3**: 使用Composition API (setup语法糖)
- **TypeScript**: 提供类型安全
- **HTML5 Canvas API**: 实现绘图功能的核心技术

### 关键组件
- **Canvas元素**: HTML5提供的绘图容器
- **Element Plus**: UI组件库，用于按钮、颜色选择器、滑块等
- **@element-plus/icons-vue**: 图标库

### Canvas API 实现原理
HTML5 Canvas 是一个二维绘图API，通过JavaScript操作像素来实现绘图功能。主要使用的API包括：
- `getContext('2d')`: 获取2D绘图上下文
- `beginPath()`, `moveTo()`, `lineTo()`, `stroke()`: 绘制线条
- `arc()`, `strokeRect()`: 绘制圆形和矩形
- `getImageData()`, `putImageData()`: 获取和恢复图像数据（用于撤销/重做和形状预览）
- `toDataURL()`: 将画布内容转换为图片数据URL

## 实现思路与流程

### 1. 初始化画布
```typescript
onMounted(() => {
  const canvas = canvasRef.value
  if (!canvas) return
  
  const ctx = canvas.getContext('2d')
  if (!ctx) return
  
  // 设置白色背景
  ctx.fillStyle = '#FFFFFF'
  ctx.fillRect(0, 0, canvas.width, canvas.height)
  
  saveHistory() // 保存初始状态到历史记录
})
```

### 2. 绘图工具实现

#### 画笔/橡皮擦工具
- 监听鼠标事件：`mousedown`, `mousemove`, `mouseup`, `mouseleave`
- 在`mousedown`时开始路径，设置样式
- 在`mousemove`时连续绘制线条
- 橡皮擦实际上是用白色绘制

```typescript
const startDrawing = (e: MouseEvent) => {
  isDrawing = true
  const [x, y] = getCoordinates(e)
  startX = x
  startY = y
  
  const ctx = canvasRef.value!.getContext('2d')!
  snapshot = ctx.getImageData(0, 0, canvasWidth.value, canvasHeight.value)
  
  ctx.beginPath()
  ctx.moveTo(x, y)
  ctx.strokeStyle = currentTool.value === 'eraser' ? '#FFFFFF' : strokeColor.value
  ctx.lineWidth = strokeWidth.value
  ctx.lineCap = 'round'
  ctx.lineJoin = 'round'
}

const draw = (e: MouseEvent) => {
  if (!isDrawing) return
  
  const canvas = canvasRef.value!
  const ctx = canvas.getContext('2d')!
  const [x, y] = getCoordinates(e)
  
  if (currentTool.value === 'pen' || currentTool.value === 'eraser') {
    ctx.lineTo(x, y)
    ctx.stroke()
  }
  // ... 其他工具
}
```

#### 几何形状工具（直线、矩形、圆形）
- 使用`snapshot`保存按下鼠标时的画布状态
- 在拖动过程中，先恢复snapshot，再绘制新形状
- 实现预览效果，避免留下拖动轨迹

```typescript
} else if (currentTool.value === 'line') {
  ctx.putImageData(snapshot!, 0, 0) // 恢复快照
  ctx.beginPath()
  ctx.moveTo(startX, startY)
  ctx.lineTo(x, y)
  ctx.stroke()
} else if (currentTool.value === 'rect') {
  ctx.putImageData(snapshot!, 0, 0)
  ctx.strokeRect(startX, startY, x - startX, y - startY)
} else if (currentTool.value === 'circle') {
  ctx.putImageData(snapshot!, 0, 0)
  ctx.beginPath()
  const radius = Math.sqrt(Math.pow(x - startX, 2) + Math.pow(y - startY, 2))
  ctx.arc(startX, startY, radius, 0, Math.PI * 2)
  ctx.stroke()
}
```

### 3. 撤销/重做机制
使用ImageData数组保存历史记录：

```typescript
const history = ref<ImageData[]>([])
const historyIndex = ref(-1)

const saveHistory = () => {
  const canvas = canvasRef.value
  if (!canvas) return
  
  const ctx = canvas.getContext('2d')
  if (!ctx) return
  
  // 移除重做状态
  history.value = history.value.slice(0, historyIndex.value + 1)
  
  history.value.push(ctx.getImageData(0, 0, canvasWidth.value, canvasHeight.value))
  historyIndex.value++
  
  // 限制历史记录数量为50
  if (history.value.length > 50) {
    history.value.shift()
    historyIndex.value--
  }
}

const undo = () => {
  if (historyIndex.value <= 0) return
  historyIndex.value--
  restoreHistory()
}

const redo = () => {
  if (historyIndex.value >= history.value.length - 1) return
  historyIndex.value++
  restoreHistory()
}

const restoreHistory = () => {
  const canvas = canvasRef.value
  const ctx = canvas?.getContext('2d')
  if (!canvas || !ctx || history.value.length === 0) return
  
  ctx.putImageData(history.value[historyIndex.value], 0, 0)
}
```

### 4. 移动端支持
通过触摸事件转换实现：

```typescript
const startDrawingTouch = (e: TouchEvent) => {
  const touch = e.touches[0]
  const mouseEvent = new MouseEvent('mousedown', {
    clientX: touch.clientX,
    clientY: touch.clientY,
  })
  startDrawing(mouseEvent as any)
}
```

### 5. 导出图片
```typescript
const downloadImage = () => {
  const canvas = canvasRef.value
  if (!canvas) return
  
  const link = document.createElement('a')
  link.href = canvas.toDataURL('image/png')

  link.click()
```

## 整体实现细节

### 组件结构
```
Whiteboard.vue
├── 工具栏
│   ├── 绘图工具选择（画笔、直线、矩形、圆形、橡皮擦）
│   ├── 颜色选择器（el-color-picker）
│   ├── 粗细调节（el-slider）
│   ├── 撤销/重做/清空按钮
│   └── 导出图片按钮
├── 画布区域（canvas元素）
└── 画布尺寸选择
```

### 状态管理
- `currentTool`: 当前选中的工具
- `strokeColor`: 画笔颜色
- `strokeWidth`: 画笔粗细
- `isDrawing`: 是否正在绘制
- `history`: 历史记录数组
- `historyIndex`: 当前历史索引

### 坐标计算
```typescript
const getCoordinates = (e: MouseEvent): [number, number] => {
  const canvas = canvasRef.value!
  const rect = canvas.getBoundingClientRect()
  return [e.clientX - rect.left, e.clientY - rect.top]
}
```

### 画布尺寸调整
支持三种预设尺寸：800x600, 1200x800, 1920x1080
调整时会清空画布和历史记录

## 功能总结

### 核心特性
1. **多种绘图工具**: 支持画笔、直线、矩形、圆形、橡皮擦
2. **自定义样式**: 可调节颜色和粗细
3. **撤销/重做**: 最多保存50步历史记录
4. **跨平台支持**: 同时支持鼠标和触摸操作
5. **图片导出**: 一键导出PNG格式图片
6. **响应式设计**: 使用Tailwind CSS实现响应式布局

### 技术亮点
- 纯前端实现，无需后端支持
- 使用Canvas API直接操作像素，性能优秀
- 通过ImageData实现撤销/重做功能
- 触摸事件转鼠标事件，简化移动端适配
- 使用Vue 3 Composition API，代码组织清晰

### 优化空间
1. 可以添加图层功能
2. 支持更多形状和自定义路径
3. 添加文字输入功能
4. 支持导入图片作为背景
5. 添加更多的导出格式（如JPG、SVG）
